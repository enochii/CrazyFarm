# Crazy Farm

## HLR代码规范

- 类内变量请以单下划线开头
- 修饰词以`public`在`abstract`前面
- 一个模式如果和其他package内的类关系不大，新建一个package，参见`Adapter`
- 简单的测试函数可以写在类内的static main()中
